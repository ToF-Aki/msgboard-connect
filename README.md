# NEC Solution Innovators 50周年記念メッセージボード

## 概要
NEC Solution Innovatorsの50周年記念を祝うオンラインメッセージボードアプリです。

## 機能
- **管理画面**: メッセージの管理、タイトル変更、背景設定
- **ユーザー投稿**: フォント・色選択、140文字制限
- **ユーザー閲覧**: 浮遊アニメーション、ループ表示
- **全画面表示**: 緑背景でのカード表示

## 技術スタック
- Flutter Web
- Provider (状態管理)
- Google Fonts
- File Picker

## デプロイ
Vercelでデプロイされています。

## 運用URL
- 管理画面: `/admin`
- ユーザー投稿: `/post`
- ユーザー閲覧: `/user-view`# Trigger workflow
